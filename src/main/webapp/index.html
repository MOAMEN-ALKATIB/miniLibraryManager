Library System

Project Idea: This project is a simple Library Management System built using Java REST API. It allows users to manage books including adding, deleting, searching, and sorting. The system is connected to a database and exposes endpoints using JAX-RS. The main goal of this project is to practice: OOP, REST API design, JDBC, and clean architecture.

⚙️ Features

  • ➕ Add Book
  • ❌ Delete Book
  • 🔍 Find Book by ISBN
  • 🔎 Find Book by Title
  • 📚 Display All Books
  • 👤 Display Books by Author
  • 📊 Sort Books by Year
  • 🔤 Sort Books by Title
  • Add Book
  • Display All Books
  • Search Books
